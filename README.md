# VIT Teacher Skills

A collection of skills for VIT-registered teachers in Victoria, Australia — covering the full professional lifecycle: registration (Inquiry, renewal, professional conduct), daily teaching practice (lesson planning, assessment, reporting), student support (IEPs, behaviour, incidents), and parent communication. All aligned to the Australian Professional Standards for Teachers (APST), the Victorian Curriculum F–10 Version 2.0, and VCE Study Designs.

## Skills Included

### Registration & Career Skills

| Skill | What it does | Use frequency |
|---|---|---|
| `vit-inquiry` | PRT → Full registration: 5-stage Inquiry, all 37 APST descriptors at Proficient level | Once per career |
| `vit-registration-renewal` | Annual registration renewal: PL hours, recency of practice, MyVIT process | Annually |
| `vit-professional-conduct` | VIT Codes of Conduct and Ethics, mandatory notifications, professional boundaries | As needed |
| `professional-learning-reflection` | Annual 20-hour PL documentation, APST domain coverage, VIT audit-ready | Ongoing/annually |

### Teaching Practice Skills

| Skill | What it does | Use frequency |
|---|---|---|
| `unit-planning` | Whole-unit design, scope & sequence, curriculum mapping | Termly |
| `lesson-planning` | Single lesson plans aligned to Victorian Curriculum / VCE — explicit instruction, differentiation, APST-aligned | Daily |
| `assessment-design` | Assessment tasks, rubrics, feedback frameworks | Fortnightly |
| `report-card-writing` | Student report comments — 5-point scale, plain English | Twice yearly |

### Student Support & Communication Skills

| Skill | What it does | Use frequency |
|---|---|---|
| `iep-writing` | IEPs, ILPs, NCCD documentation, EAL/high-ability plans | Ongoing |
| `behaviour-support-planning` | Proactive BSPs, functional behaviour assessments, SWPBS | Ongoing |
| `incident-documentation` | Incident reports, mandatory reporting (CYFA s.184), child safety | As needed |
| `parent-communication` | Parent emails, newsletters, interviews, complaints — APST 7.3 | Daily/weekly |

## Installation

Install all 12 skills at once:

```bash
npx skills@latest add luojiahai/vit-teacher-skills --all
```

Or pick individual skills interactively:

```bash
npx skills@latest add luojiahai/vit-teacher-skills
```

## Requirements

These skills are designed for:
- VIT-registered teachers in Victoria, Australia
- Teachers working with the Victorian Curriculum F–10 or VCE
- School leaders, mentors, and CRTs working in Victorian schools
- Provisionally registered teachers (PRTs) completing the VIT Inquiry

## Alignment to Professional Standards

Every skill in this collection aligns to the **Australian Professional Standards for Teachers (APST)**:

| Domain | Skills primarily aligned |
|---|---|
| **Professional Knowledge** (Standards 1–2) | `lesson-planning`, `unit-planning`, `iep-writing` |
| **Professional Practice** (Standards 3–5) | `assessment-design`, `report-card-writing`, `behaviour-support-planning`, `incident-documentation` |
| **Professional Engagement** (Standards 6–7) | `professional-learning-reflection`, `parent-communication`, `vit-inquiry`, `vit-registration-renewal`, `vit-professional-conduct` |

## Victorian Context

Skills reference:
- **VIT** — Victorian Institute of Teaching regulatory framework
- **APST** — Australian Professional Standards for Teachers (as assessed by VIT)
- **Victorian Curriculum F–10 Version 2.0** — all 8 learning areas and 4 capabilities
- **VCE Study Designs** — all subject groups and assessment types
- **VTLM 2.0** — Victorian Teaching and Learning Model, explicit instruction
- **DET Reporting Policy** — 5-point age-related scale, mode-level English, Mathematics v2.0
- **SWPBS Framework** — School-Wide Positive Behaviour Support
- **Child Safety** — Four Critical Actions, mandatory reporting (CYFA s.184, Crimes Act s.327)
